# Architecture Portfolio

A modern portfolio application showcasing system architecture case studies, built with React and TypeScript.

## Tech Stack

### Frontend Framework
- **React 19** - UI library for building the interface
- **TypeScript** - Type-safe JavaScript for better development experience
- **Vite 6** - Fast build tool and development server

### Routing
- **React Router DOM 7** - Client-side routing for navigation between pages

### Styling
- **Tailwind CSS 4** - Utility-first CSS framework for styling
- **@tailwindcss/typography** - Plugin for beautiful typography styling
- **PostCSS** - CSS processing with Autoprefixer

### Data Management
- **TanStack Query (React Query)** - Server state management and data fetching
- **Contentful** - Headless CMS for content management
  - GraphQL API for fetching case studies and content
  - Rich Text rendering with `@contentful/rich-text-react-renderer`

### Development Tools
- **ESLint** - Code linting and quality checks
- **TypeScript ESLint** - TypeScript-specific linting rules

### Testing
- **Vitest** - Fast unit test framework
- **React Testing Library** - Component testing utilities
- **@vitest/coverage-v8** - Code coverage reporting
- **happy-dom** - Lightweight DOM implementation for testing
